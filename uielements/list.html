List

Lists present multiple line items vertically as a single continuous element. Lists are best suited for similar data types.

States

Every item in a list should:

  • Lists should be sorted in logical ways that make content easy to scan.
  • Start with a capital letter (be written in sentence case).
  • Not use commas or semicolons at the end of each line.
